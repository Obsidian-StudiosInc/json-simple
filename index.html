Download the JSON.simple 2.0.0 release jar, javadocs, and source.
As a maven dependency in a java 7+ project:
<dependency>
    <groupId>com.github.cliftonlabs</groupId>
    <artifactId>json-simple</artifactId>
    <version>2.0.0</version>
</dependency>
Usage:
/* Quickly serialize stuff in json-simple (will throw IllegalArgumentException if stuff cannot be serialized into JSON): */
Object stuff = null;            
String serialized = Jsoner.serialize(stuff);

/* Deserialize stuff in json-simple by handling a DeserializationException: */
try{
    Object deserialized = Jsoner.deserialize(serialized);
}catch(DeserializationException caught){
    /* Oops bad JSON. */
}

/* When you know you only want a JsonObject or an empty one if it fails: */
JsonObject deserializedObject = Jsoner.deserialize(serialized, new JsonObject());

/* When you know you only want a JsonArray or an empty one if it fails: */
JsonArray deserializedArray = Jsoner.deserialize(serialized, new JsonArray());

/* When you want it to look pretty: */
String prettyPrinted = Jsoner.prettyPrint(serialized);

Define how your objects are serialized as JSON by implementing the Jsonable interface.

JsonArray and JsonObject extend ArrayList and HashMap respectively and have typed getters for each kind of JSON value.

Browse the JSON.simple 2.0.0 release javadocs for more details.
